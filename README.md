# Python Task Manager

## Description

This is a terminal-based Task Manager application built using Python.

The program allows users to:
- Add tasks
- View tasks
- Remove tasks
- Save tasks permanently using file storage
- Load saved tasks automatically when the program starts

This project was built as part of my Python learning journey while practicing software development fundamentals.

---

## Features

- Persistent task storage using `.txt` files
- Automatic loading of saved tasks
- Input validation
- Remove tasks by index
- Organized using Python functions
- Simple terminal interface

---

## Technologies Used

- Python 3
- File Handling
- Functions
- Lists
- Loops
- Conditional Logic

---

## How to Run

Run the Python file using:

```bash
python task_manager_app.py
```
